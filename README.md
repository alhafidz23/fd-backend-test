# fd-backend-test

== Running application local

- Create database name "DB_FD"
- Import t_order.sql to database
- Open directory script with command line (cmd/visual code)
- Install package with command "npm i"
- Run script with command "npm run female"
- Application running on port 3010
- To open data with table open data.html

== Running application hosting

- Create database name "DB_FD"
- Import t_order.sql to database
- Open directory script
- Install package with command "npm i"
- Run script with command PM2 application "pm2 start app.js --name='api'"
- API running on port 3010
